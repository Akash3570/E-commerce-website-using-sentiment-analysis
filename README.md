# E-commerce Website using Sentiment Analysis

## Features
- Product recommendation based on sentiment analysis
- BERT model integration
- Django-based web application

## Setup Instructions
1. Clone the repo
https://github.com/Akash3570/E-commerce-website-using-sentiment-analysis

2. Install dependencies
pip install -r requirements.txt

3. Run server
python manage.py runserver

## Note
Dataset and model files are not included due to size limits.
